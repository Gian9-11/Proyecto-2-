# Simulador de Cola de Impresión con Montículo Binario

## Descripción
Aplicación en Java que simula un sistema operativo de gestión de colas de impresión mediante un montículo binario (min-heap) como cola de prioridad, permitiendo administrar usuarios, documentos y prioridades de impresión, con soporte de tablas de dispersión para búsqueda eficiente O(1) de documentos en cola.
